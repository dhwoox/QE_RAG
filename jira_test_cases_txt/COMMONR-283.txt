■ 이슈 키: COMMONR-283
■ 제목: Smartcard byte order 설정
■ 설명: BDP-511

BioStar2 v2.8.16 에 해당기능 지원

 

*1.1.1.3.2 iClass 카드들은 Out 되는 값을 Card ID 로 사용 시, byte order 설정에 영향을 받지 않으며 PAC data 그대로 출력한다.
 * 1.1.1.3.2.1 단, User ID 를 Out 값으로 설정 시, User ID 가 출력되고 byte order 설정에 영향을 받는다.
============================================================

■ 테스트 스텝

스텝 (Step)
