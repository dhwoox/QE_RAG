■ 설명: 재현 환경 구성

!image-2024-07-17-14-54-25-208.png!

 

: 로컬 망 구성(외부망 LAN선은 공유기에 연결하지 않은 상태로 환경구성)
 => PC가 공유기에 연결 - 공유기에 허브가 연결 - 허브에 장치 연결된 상태
 : 공유기의 Lease time(DHCP 임대 시간)을 1분으로 설정
 : 위의 환경 구성으로 바이오스타에 장치 추가해 사용중

[Step 1]
스텝 (Step)
[본문의 환경으로 구성된 상태]
1) (장치 연결된 상태에서) 1분 이후 공유기와 허브간 연결된 랜선 제거
2) 네트워크 연결 해제 확인 후, 랜선 재연결
3) 바이오스타와의 통신유무 확인

데이터 (Data)
서버 연결 
1. Server to Device Mode
2. DHCP

예상 결과 (Expected Result)
1. DHCP Lease Time 이후 공유기와 허브간 랜선 제거 후 네트워크 연결 해제 시 랜선을 재연결하여 자동으로 BioStar와 연결이 되는 지 확인한다.
