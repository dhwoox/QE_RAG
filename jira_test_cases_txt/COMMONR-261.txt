■ 이슈 키: COMMONR-261
■ 제목: BioStar에서 Mobile Access 장치등록 지원
■ 설명: XPD2 Rev. / XP2 V1.2.0 / N2 V1.3.0 지원

바이오 스타> 설정> 모바일 액세스에서 설정 지원

Test 전 환경설정(1)
1) https://mc.suprema.io에 접속
2) 사이트 등록 후 Site&License에서 Site ID 확인
    => BioStar2에서 모바일 크리덴셜 설정시 Site ID에 들어가는 값임.
3) 모바일 크리덴셜 카드 추가
4) 모바일로 이메일 접속해 모바일 크리덴셜  App Link 클릭해 모바일 크리덴셜 카드 등록

Test 전 환경설정(2)
1) 사용자> Add> +모바일 클릭
2) 카드ID 입력> 등록
3) 이메일 주소 입력 후 확인
4) 해당 사용자를 장치에 전송
============================================================

스텝 (Step)
1. 바이오스타 접속
2. 설정> Mobile Access
3. 각 항목들 설정
4. 등록용 장치설정 - 추가 클릭

데이터 (Data)
각 항목 설정

: 이메일/암호 - (ex) 회사이메일/암호
: 사이트ID - Test 전 환경설정 참고

예상 결과 (Expected Result)
