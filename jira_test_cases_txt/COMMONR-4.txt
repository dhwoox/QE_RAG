■ 이슈 키: COMMONR-4
■ 제목:  HID Prox Card By Wiegand Format Check
■ 설명: - HID Prox 인 FSK-1 ~ FSK-63 카드 동작 확인

- HID Prox 카드에 대해 Wiegand Format 전달받지 못하였기에 ID Field만 각 포맷에 맞게 설정 후 진행

- 신규 제품으로 한해 검증진행
============================================================

■ 테스트 스텝

[Step 1]
스텝 (Step)
1. Setting> Card Format> Wiegand
2. Custom Wiegand Format 생성
3. Device> Wiegand Format> HID Prox 카드가 지원하는 포맷설정
4. 카드Read

데이터 (Data)
HID Prox 카드 63장
: FSK1~FSK63

Custom Wiegand Format
: HID Prox 카드의 Format으로 생성

예상 결과 (Expected Result)
카드의 Raw Data 값이 읽힌다.
