■ 이슈 키: COMMONR-369
■ 제목: [FISF-2343] FSF2-TCM10 사용 중 얼굴 인증 화면이 붉게 출력되는 현상에 대한 재발방지대책
============================================================

■ 테스트 스텝

[Step 1]
스텝 (Step)
1. 메뉴> 장치> 열화상 카메라 설정
2. 인증시도> Card 인증> 온도 체크 시 Threshold 초과하는 곳으로 위치 시킨 후, 입력 안하고 대기> 실패 시 결과 확인

데이터 (Data)
열화상 카메라 설정
- 사용(하드)
- 온도확인모드
=> 인증 후 온도확인

예상 결과 (Expected Result)
1. 재현 과정에 붉은색이 일시적으로 출력되나, 1.5초 후 다시 정상화면으로 전환된다.
------------------------------------------------------------
[Step 2]
스텝 (Step)
1. 메뉴> 장치> 열화상 카메라 설정
2. 인증시도> Threshold 초과하는 곳으로 위치 시킨 후, 얼굴 입력하여 온도 체크 시도> 여러 번 수행> 결과 확인

데이터 (Data)
열화상 카메라 설정
- 사용(하드)
- 온도확인모드
=> 체크만

예상 결과 (Expected Result)
1. 재현 과정에 붉은색이 일시적으로 출력되나, 1.5초 후 다시 정상화면으로 전환된다.
------------------------------------------------------------
[Step 3]
스텝 (Step)
1. 메뉴> 장치> 열화상 카메라 설정
2. 인증시도> Threshold 초과하는 곳으로 위치 시킨 후, 얼굴 입력하여 온도 체크 시도> 여러 번 수행> 결과 확인

데이터 (Data)
마스크 설정
- 사용(하드 / 소프트)
열화상 카메라 설정
- 사용(하드)
- 온도확인모드
=> 인증 후 온도확인, 체크만
스크린 세이버 설정
- 사용 / 미사용

예상 결과 (Expected Result)
1. 재현 과정에 붉은색이 일시적으로 출력되나, 1.5초 후 다시 정상화면으로 전환된다.
