■ 이슈 키: COMMONR-247
■ 제목: [BSPP-412] Batch Edit에서 장치 Admin 추가방식 변경관련 검증TC
============================================================

■ 테스트 스텝

[Step 1]
스텝 (Step)
1. 장치 두대이상 선택> Batch Edit 클릭
2. 다수의 관리자(All/User/Config) 최대로 설정 후 적용
3. 장치상세정보창 진입
4. 관리자 확인
5. All/User/Config 관리자로 메뉴진입시도

데이터 (Data)
> Max 관리자수: 1000명
> 각 장치별 관리자 미설정된 상태

예상 결과 (Expected Result)
1. All/User/Config 관리자가 설정된다.
: 장치상세정보창_관리자 항목에 표시
2. 관리자로 메뉴진입시 각 관리자에 해당하는 메뉴에 접근이 가능해야 한다.
------------------------------------------------------------
[Step 2]

2. All/User/Config 관리자를 1명씩 설정 후 적용
3. 장치상세정보창 진입
4. 관리자 확인
5. All/User/Config 관리자로 메뉴진입시도

데이터 (Data)
> Max 관리자수: 1000명
> 각 장치별 관리자 Max로 설정된 상태

예상 결과 (Expected Result)
1. 이미 Max로 설정되었기 때문에 새로 내린 관리자는 저장되지 않는다.
2. 기존 관리자로 메뉴진입이 가능하다.

2. 기 설정된 관리자 & 새 관리자로 메뉴진입이 가능하다.
